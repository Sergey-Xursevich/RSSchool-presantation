What is React.js

  • React is a JavaScript library that is used to create a user interface.
  • React was originally intended for websites, for creating websites, but later React Native appeared, which is already designed for mobile devices.
  • React to the ideal tool for building scalable web applications.

In total

  1. React is designed to create the fastest possible applications.
  2. The “unit” of an application on React is a component.
  3. The component is something complete and contains all the markup and logic necessary for the component to exist outside of any application.
  4. Any component is controlled by the state, which, when changed, renders the component.
